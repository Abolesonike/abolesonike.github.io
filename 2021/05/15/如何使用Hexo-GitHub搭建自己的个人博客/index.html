二、安装hexo框架

1.使用cnpm安装hexo

查看hexo版本

2.使用hexo生成博客项目

在自己喜欢的位置创建一个文件夹，博客中所有的文件都会保存在这个文件夹中。如果中通出现错误，或者想重来，直接删除该文件夹就可以了。

进入文件夹，输入命令，生成博客并初始化

该命令会去GitHub上克隆所需要的文件，完成后可以看见文件夹下多出了许多文件

3.启动本地博客服务

启动完成后在本地4000端口就可以预览博客了

三、将博客托管到GitHub

1.登录GitHub并新建一个仓库

注意仓库的名字必须是yourGitHubName.github.io

2.在blog目录下安装git部署插件

3.配置_config.yml

在博客根目录下修改**_config.yml** 配置文件

4.部署到Github仓库里

此过程需要输入你GitHub的账号跟密码

最后通过https://YourGithubName.github.io 这个地址就可以访问到博客了

四、修改主题

hexo支持众多主题，大家可以在网上搜索下载自己喜欢的主题，这里以yilia主题为类。

一个简洁优雅的hexo主题 A simple and elegant theme for hexo. (github.com)

1.安装

2.配置

修改hexo根目录下的 _config.yml ： theme: yilia

3.更新

查看博客就可以看见主题生效了

五、hexo常用命令

hexo s #启动本地博客服务

hexo clean #清理

hexo g #生成

hexo n 我的第一篇文章 #创建新的文章

hexo d #部署到Github仓库里
